Задание: «Справочник организаций»

Необходимо создать программу, реализующую работу со справочником, где каждая запись хранит информацию об организации.

1. Поля записи (класс Client):
Название фирмы
Владелец
Телефон
Адрес
Род деятельности

2. Требуется реализовать следующие возможности (класс Directory):
Поиск по названию фирмы
Поиск по владельцу
Поиск по номеру телефона
Поиск по роду деятельности
Просмотр всех записей
Добавление новых записей

3. Хранение данных:
Вся информация хранится в текстовом файле с возможностью дозаписи новых данных без удаления предыдущих.
4. Структура программы:
Класс Client — описывает одну запись (фирму).
Класс Directory — управляет массивом объектов Client, выполняет поиск, добавление и вывод.

5. Основные действия:
При запуске программа считывает данные из файла.
Пользователь может добавить запись, выполнить поиск или вывести весь список.
После завершения работы новые данные сохраняются в файл.
